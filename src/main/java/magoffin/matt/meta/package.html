sMeta: simple metadata API for Java

The sMeta library provides a simple way to extract metadata from various file types. To start, look to the {@link magoffin.matt.meta.MetadataResourceFactoryManager} class. This is the main entry point to the library, and provides a way to look up {@link magoffin.matt.meta.MetadataResourceFactory} instances that are capable of instantiating {@link magoffin.matt.meta.MetadataResource} instances for extracting the actual metadata.

In general, the steps for extracting metadata for a given file are thus:

  1. Get a {@link magoffin.matt.meta.MetadataResourceFactoryManager} via {@link magoffin.matt.meta.MetadataResourceFactoryManager#getDefaultManagerInstance()}.
  2. Use the manager to locate a {@link magoffin.matt.meta.MetadataResourceFactory} for your file via {@link magoffin.matt.meta.MetadataResourceFactoryManager#getMetadataResourceFactory(java.io.File)}
  3. Use the returned factory to instantiate a {@link magoffin.matt.meta.MetadataResource} for this file via {@link magoffin.matt.meta.MetadataResourceFactory#getMetadataResourceInstance}.
  4. Now you have access to the available metadata via the methods of the {@link magoffin.matt.meta.MetadataResource} interface.
