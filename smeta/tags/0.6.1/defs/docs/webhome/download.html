sMeta: download

You can download sMeta for free; it will run on any platform with a Java 5 runtime.

Requirements

sMeta is a 100% Java library. Currently the requirements are as follows:

Java 5
Get the appropriate Java SDK/JDK for your server platform. You can get this from Sun, IBM, or Blackdown.

Download

You can download sMeta from SourceForge. The binary package (named like smeta-1.0.0.tbz2) contains the sMeta JAR along with instructions for getting your installation going for the first time. The source package (named like smeta-1.0.0-src.tbz2) contains all of sMeta source code.

For installation instructions look in the README file that comes in each distribution.
