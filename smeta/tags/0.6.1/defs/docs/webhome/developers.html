Developers

Developers, developers, developers!

There are many ways to contribute to the development of sMeta, including Java, HTML/CSS, documentation, and localization.

sMeta is designed so that additional metadata formats can be easily added. If you have a metadata format that is not currently supported, you can help out my implementing the classes to support it and donating that to the sMeta project.

sMeta also benefits from many test cases. If you find you have a sMeta support file that is not being parsed correctly, please donate that file to sMeta so it can be added as a test case (no copyrighted material, please).

If you're interested in helping out, please email msqr@users.sourceforge.net.
